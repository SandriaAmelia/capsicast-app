# capsicast-app 🌶✨
Forecasting Large Red Chili Prices in East Java Using a GSTAR-SUR Model 

## Setup environment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install numpy pandas scipy statsmodels linearmodels scikit-learn matplotlib seaborn geopy streamlit
```

## Run steamlit app
```
streamlit run capsicast.py
```
